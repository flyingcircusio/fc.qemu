fc.qemu development
===================

Prepare working copy in a personal box::

    host$ hg clone https://bitbucket.org/flyingcirucs/fc.qemu
    host$ cd fc.qemu
    host$ vagrant up
    host$ vagrant ssh

Run the tests::

    vm$ cd /vagrant
    vm$ sudo bin/py.test

Test execution automatically updates a coverage report in the `htmlcov`
directory.
