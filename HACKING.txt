fc.qemu development
===================

Prepare working copy in a personal box::

    host$ hg clone https://bitbucket.org/flyingcirucs/fc.qemu
    host$ cd fc.qemu
    host$ vagrant up
    host$ vagrant ssh
    vm$ cd /vagrant
    vm$ sudo ./bootstrap.sh

Run the tests::

    vm$ sudo bin/py.test
